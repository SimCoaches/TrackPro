- **OS**: Windows 10/11 (64-bit)

- **Storage**: 500MB free space

### Installation Steps

1. **Download** the latest installer from the [Releases](https://github.com/simcoaches/trackpro/releases) page
2. **Run** the installer as Administrator
3. **Follow** the installation wizard - it will automatically install required prerequisites:
   - vJoy (virtual joystick driver)
   - HidHide (device isolation)
   - Visual C++ Redistributables
4. **Launch** TrackPro from your desktop or start menu

### First Time Setup
1. **Create Account**: Sign up with email or OAuth provider
2. **Verify Phone**: Enable SMS 2FA for security
3. **Calibrate Pedals**: Set up your pedal hardware
4. **Connect iRacing**: Start recording your racing sessions

## ğŸ”§ Troubleshooting

### Common Issues

**TrackPro won't start**
- Ensure all prerequisites are installed
- Run as Administrator
- Check Windows Firewall settings

**No telemetry data**
- Verify iRacing is running
- Check TrackPro has access to iRacing data
- Restart both applications

**Voice chat not working**
- Check microphone permissions
- Verify internet connection
- Test audio settings

**Pedal calibration issues**
- Ensure vJoy is properly installed
- Check device permissions
- Try recalibration
